01

תקציר ההצעה

Executive Summary

הצעה זו מפרטת את עלות פיתוח גרסת MVP של פלטפורמת "מלאך הזכויות" — מערכת SaaS לאיתור, חישוב, והגשה פרו-אקטיבית של זכויות. הפלטפורמה מבוססת על שיחת AI טבעית בעברית, זמינה 24/7, וכוללת סוכן פרו-אקטיבי אוטומטי מאושר סופית על ידי אדם אנושי (human-in-the-loop) — כל פעולה משמעותית (שליחת מכתב, ערעור, פרסום זכות חדשה) מחייבת אישור אנושי מפורש לפני שהיא יוצאת החוצה. הסוכן מטפל בהגשה, מעקב, וערעורים עד למימוש מלא של הזכות.

התיחום של MVP זה כולל שתי זכויות, בהתאם לבקשה שהוגשה לרשות החדשנות:
(1) הנחת ארנונה למילואים פעיל — זכות מאוחדת ב-2 רמות-tier (חייל 5% · מפקד 25% על 100 מ"ר ראשונים).
(2) הנחת ארנונה לאזרחים ותיקים — הנחה מדורגת לפי גיל, הכנסה ומצב משפחתי (עד 100%).
שתי הזכויות מיועדות ל-3 רשויות מקומיות מובילות (תל אביב, ירושלים, חיפה). מעל הזכויות נבנית תשתית "מנוע שמחולל מנועי זכות" (Right Forge) — מערכת שמייצרת זכויות חדשות אוטומטית באמצעות Claude Code AI, מתוך מקורות טקסטואליים (כל זכות, חוקים, תקנות).

03

תוצרי הפרויקט

Deliverables

בסיום הפיתוח, הלקוח יקבל את התוצרים הבאים:

01

פלטפורמה חיה ב-Production

סביבת Production רצה ומאובטחת.

02

2 זכויות ב-DSL: ארנונה-מילואים-פעיל (חייל + מפקד) · ארנונה-אזרחים ותיקים

שתי RightDefinitions: (א) זכות מאוחדת מילואים-פעיל עם 2 tiers · (ב) זכות אזרחים ותיקים מדורגת. שתיהן חתומות דיגיטלית ע"י עו"ד מוסמך, נשמרות ב-PostgreSQL.

03

DSL Engine · מנוע גנרי

Interpreter שיודע להריץ כל RightDefinition: בדיקת זכאות, חישוב, מסמכים, מכתב, lifecycle, ערעור. נכתב פעם אחת — משרת לנצח.

04

Right Forge · מנוע שמחולל מנועים

ממשק אדמין + שירות שרת עם Claude Code AI: מזין URL מ"כל זכות" → ה-AI מייצר טיוטת RightDefinition → אדמין סוקר ומאשר → הזכות חיה במערכת.

05

Digital Twin Layer

תאום דיגיטלי שמרכז את כל פרטי המשתמש (אישיים, נכס, מילואים, כלכלי) ומועבר אוטומטית בין כל זכויות הפלטפורמה לבדיקת זכאות.

06

Proactive Layer · התראות פרו-אקטיביות

Cron + Watchers ברקע · התראות In-App, Email, הכנה ל-SMS.

07

Authority Adapters · 3 ערים

אוטומציית הגשה ייעודית לעיריות תל אביב, ירושלים, חיפה — מיילים רשמיים לגזברויות + תבניות מכתב.

08

פאנל ניהול אדמין · עורך DSL + ניהול רשויות

UI אדמין: עורך DSL JSON עם validation, ניהול 255 רשויות מקומיות (CBS), חתימות עו"ד, ניהול משתמשים, ביקורת.

09

פיילוט 14 יום + הדרכה

14 ימי פיילוט פעיל בסיום הפיתוח עם תיקוני fine-tuning במסגרת התיחום, ללא עלות. 4 שעות הדרכה ל-Operators.
